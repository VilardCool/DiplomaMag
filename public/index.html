Image compression platform


This platform created for image compression with clusterization and enchantment with neural network.

First of all, choose file by pressing coresponding button, it must be in PNG or JPEG format.

Then press "Upload" button. By doing that, selected image appear in assigned area.

Then, algorithm options will appear to the right side. Select algoritm, enter parameters and press "Proceed".

Waiting flag appear in the right window and when result image is ready, it can be downloaded.
